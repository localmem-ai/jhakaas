---

- **Lighting Effects**:

### 2.3 Interactive AR

- **Head Tracking**:

## Technical Requirements by Effect Category

### Real-Time Effects (AR)
- **Tech**: MediaPipe, TensorFlow Lite, ARCore/ARKit
- **Performance**: 30-60 FPS on mobile
- **Deployment**: Mobile SDK + Web (WebGL/WebGPU)

### AI Transformation Effects (Cloud)
- **Tech**: InstantID, ControlNet, LoRAs, ComfyUI workflows
- **Performance**: 5-15 seconds per image
- **Deployment**: Cloud Run GPU instances

### Photo Enhancement
- **Tech**: Real-ESRGAN, CodeFormer, GFPGAN
- **Performance**: 2-5 seconds per image
- **Deployment**: Cloud Run or Edge (lighter models)

### Video Effects
- **Tech**: Stable Video Diffusion, EbSynth, RAFT
- **Performance**: 30-60 seconds per 5-second clip
- **Deployment**: Cloud Run GPU (higher memory)
